# GLUONiCA

Проект на базе вакуумно люминесцентного дисплея дисплея GP1294AI 256X48 и ESP32 

## Возможности часов GLUONiCA
+ показывает время
  + из rtc или ntp сервера
+ веб интерфейс для настроек
  + через браузер
  + приложение для windows
  + приложение для android
+ циферблаты с переключаемой информационной частью
+ выводит
  + температуру/влажность/СО2/давление
  + погоду используя сервис openweathermap.org
  + детектор грозы - AS3935 - нужно тестировать
+ показатели состояния пк (температура/загрузка итп)
+ будильник 100+ мелодий
+ кукушка
+ тихий режим - отключение всех шумных функций, снижение яркости по расписанию
+ есть возможно получать данные температуры/влажности/С02/пм с домашнего сервера mqtt (нужен home assistant)
+ в разработке
